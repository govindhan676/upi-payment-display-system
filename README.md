# UPI Payment Display System

## Overview
This project uses ESP32, LCD Display and Buzzer to display UPI payment notifications in real time.

## Features
- Instant payment alert
- LCD amount display
- Buzzer notification
- WiFi connectivity

## Hardware Used
- ESP32
- LCD Display
- Buzzer
- Breadboard

## Software Used
- Arduino IDE
- Embedded C

## Applications
- Shops
- Stores
- Small businesses
