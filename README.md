# 💰 UPI Payment Display System

## 📌 Overview

The UPI Payment Display System is an IoT-based smart payment notification system developed using ESP32, LCD Display, and Buzzer.

This project helps shop owners, stores, and businesses instantly verify successful UPI transactions without checking their smartphones repeatedly.

Whenever a payment is received, the ESP32 processes the payment notification and displays the received amount on the LCD display while simultaneously activating a buzzer alert.

This system provides a fast, reliable, and user-friendly solution for digital payment monitoring.

## ⚙ Working Principle

### Step 1
ESP32 connects to the WiFi network.

### Step 2
The payment notification is received from the configured payment service.

### Step 3
ESP32 extracts the payment amount.

### Step 4
The received amount is displayed on the LCD screen.

### Step 5
The buzzer generates an alert sound.

### Step 6
The shop owner verifies the payment instantly.
